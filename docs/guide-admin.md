# 책빌리지 관리자 사용 가이드

## 1. 관리자 로그인 / 가입

### 관리자 가입
1. `/admin/register`에서 가입 신청
2. 아이디, 비밀번호, 이름 입력
3. **승인 대기** 상태로 가입됨
4. 기존 관리자가 `/admin/manage`에서 **승인** 처리해야 로그인 가능

### 관리자 로그인
1. `/admin/login`에서 아이디 + 비밀번호로 로그인
2. 로그인 후 대시보드(`/admin`)로 이동

---

## 2. 대시보드 (`/admin`)

한 화면에서 도서관 현황을 파악할 수 있습니다:

- **기간별 통계**: 대출/반납/신규가입 건수
- **랭킹** (4개 탭): 다독왕 / 인기 도서 / 독서록 작성자 / 젤리 랭킹
- **연체 현황**: 현재 연체 중인 도서 목록
- **신작 도서**: 최근 등록된 도서
- **전체 도서 목록**: 간략 테이블

---

## 3. 도서 관리

### 3-1. 도서 등록 (`/admin/books/new`)

서가 관리 화면에서 서가를 선택하면 해당 서가의 도서 목록 + 등록 화면으로 이동합니다.

#### 바코드 스캔 등록 (추천)
1. **바코드 스캔** 버튼 클릭 → 카메라로 책 뒷면 바코드 스캔
2. **카카오 도서 검색 API**가 자동으로 도서 정보를 불러옵니다
3. 제목, 저자, 출판사, 소개 등이 자동 완성됩니다
4. 서재 선택 + 상세 위치 입력 후 **도서 등록** 버튼 클릭

#### ISBN 직접 입력
- 바코드 스캔이 안 되는 경우 ISBN 번호를 직접 입력하고 검색

#### 카카오에서 못 찾는 경우
- 검색 실패 시 **"이 바코드로 수동 입력하기"** 버튼이 나타납니다
- 클릭하면 바코드가 자동 입력된 상태로 수동 등록 폼 진입
- 도서명, 저자 등을 직접 입력합니다

#### ISBN 없는 도서 수동 등록
- **"ISBN 없이 수동 입력"** 버튼으로 완전 수동 등록 모드 진입
- 자체 바코드 번호 입력 + 도서 정보 수동 입력
- **표지 이미지**: 카메라 촬영 또는 사진첩에서 업로드 가능

#### 복본 (같은 책 여러 권)
- 이미 등록된 바코드를 스캔하면 **"이미 등록된 도서입니다"** 경고 표시
- 자체 바코드 스티커를 붙이고 번호를 입력하여 복본 등록

#### 개별 대여 기간
- 도서별로 기본 대여 기간과 다른 기간을 설정할 수 있습니다 (선택사항)

### 3-2. 도서 목록 (`/admin/books`)
- 전체 도서 검색 + 필터
- 도서 클릭 → 상세 정보 + 대출 이력 확인

### 3-3. 서가별 도서 (`/admin/books/shelf/[서가명]`)
- 서가 내 도서 목록 (검색, 페이지네이션)
- 도서 클릭 → 편집 다이얼로그 (제목, 저자, 위치 수정)
- 도서 삭제 (대출 중인 도서는 삭제 불가, 대출 기록 건수 안내)

### 3-4. SVG 서가 관리
- 서가 그리드를 SVG로 시각적 관리
- 서가 추가/편집/삭제, 위치/크기/색상/폰트 설정

---

## 4. 대출 처리 (`/admin/checkout`)

1. **주민 검색**: 이름 또는 동호수로 주민 검색 → 선택
2. **도서 스캔**: 바코드 스캔 또는 수동 입력
3. 대출 처리 완료 → 주민에게 젤리 +5 자동 지급

- 연체 중인 주민은 대출이 제한됩니다
- 1인당 최대 대출 권수 초과 시 대출이 제한됩니다

---

## 5. 반납 처리 (`/admin/return`)

1. **도서 스캔**: 바코드 스캔 또는 수동 입력
2. 반납 즉시 처리 → **서가 위치 팝업** 표시 (SVG 미니맵으로 꽂을 위치 안내)
3. 연체 여부 + 연체 일수 표시
4. 주민에게 젤리 +5 자동 지급
5. **반납 처리자**(어떤 관리자가 반납 처리했는지)가 자동 기록됩니다

---

## 5-1. 반납 내역 (`/admin/returns`)

- 최근 반납 완료된 도서 목록을 조회합니다 (최대 200건)
- **반납 내역 불러오기** 버튼을 클릭하여 데이터를 로드합니다
- 표시 정보: 도서명, 바코드, 대출자(이름/동호수), 반납 일시, 반납 기한, 연체 여부, 처리자
- 연체 반납 건은 빨간색 **"N일 연체"** 배지로 표시됩니다
- 20건 단위 페이지네이션

---

## 6. 연체 관리 (`/admin/overdue`)

- 현재 연체 중인 모든 도서 목록
- 각 도서별 연체 일수 표시
- **알림 발송** 버튼: 7일/30일 연체 시 알림 기록 (앱 내 알림)
- 도서명/대출자 클릭으로 상세 페이지 이동

---

## 7. 주민 관리 (`/admin/residents`)

### 주민 목록
- 전체 주민 검색 (이름, 동호수)
- 연락처는 마스킹 처리 표시

### 주민 상세 (`/admin/residents/[id]`)
- **프로필**: 이름, 동호수, 연락처
- **현재 대출 중**: 도서 목록 + 반납 기한
- **과거 대출 이력**: 반납 완료 도서 목록
- **젤리 현황**: 잔액, 이력, 수동 지급/차감

---

## 8. 랭킹 (`/admin/rankings`)

날짜 범위를 설정하여 랭킹을 조회합니다.
빠른 선택 버튼: 1주 / 1개월 / 3개월 / 6개월 / 올해

- **다독왕**: 가장 많이 대출한 주민
- **인기 도서**: 가장 많이 대출된 도서
- **독서록 작성자**: 독서록을 많이 작성한 주민
- **젤리 랭킹**: 젤리를 많이 획득한 주민

---

## 9. 설정 관리 (`/admin/manage`)

### 사이트 설정
- 아파트(도서관) 이름 설정

### 도서관 설정
- **1인당 최대 대출 권수** (기본 3권)
- **기본 대출 기간** (기본 14일)
- **신작 도서 기준 일수** / 수동 신작 유지 기간

### 젤리 포인트 설정
- 대출/반납/독서록/퀴즈별 젤리 지급량 설정

### 퀴즈 관리
- 도서별 퀴즈 등록/편집/삭제
- 객관식 문제 + 정답 설정

### 연체 관리
- 연체 설정 확인

### 관리자 승인
- 가입 신청한 관리자 승인/거절

---

## 10. 기타

### 도서 삭제 내역 (`/admin/deletions`)
- 삭제된 도서의 감사 로그 (누가, 언제, 어떤 도서를 삭제했는지)

### 대여중 도서 목록 (`/admin/rentals`)
- 현재 대출 중인 전체 도서 목록 (반납 기한 포함)

### 관리자 사이드바 메뉴 순서
대시보드 → (구분선) → 주민 목록 → 랭킹 → (구분선) → 전체 도서 → 대여중 도서 → 도서 등록 → 대출 → (구분선) → 반납 → 반납 내역 → (구분선) → 연체 내역 → 도서 삭제내역 → (구분선) → 관리
