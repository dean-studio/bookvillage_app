# 관리자 가이드

관리자 기능의 전체 사용 방법을 안내합니다.
각 기능의 상세 내용은 사이드바의 기능 페이지를 참고하세요.

## 관리자 가입 / 로그인

### 가입

1. `/admin/register`에서 가입 신청
2. 아이디, 비밀번호, 이름 입력
3. **승인 대기** 상태로 가입됨
4. 기존 관리자가 설정에서 **승인** 처리해야 로그인 가능

### 로그인

1. `/admin/login`에서 아이디 + 비밀번호로 로그인
2. 로그인 후 대시보드로 이동
