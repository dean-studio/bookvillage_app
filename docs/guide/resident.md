# 주민 가이드

주민 기능의 전체 사용 방법을 안내합니다.

## 회원가입 / 로그인

### 첫 이용 (회원가입)

1. 앱에 접속하면 로그인 화면이 나타납니다
2. **이름** → **휴대폰 번호** → **동** → **호** → **PIN 4자리** 순서로 입력
3. 모든 숫자는 화면의 큰 숫자패드로 입력합니다
4. 각 단계에서 "이전" 버튼으로 돌아갈 수 있습니다

### 로그인

1. **휴대폰 번호** 입력
2. **PIN 4자리** 입력
3. 자동으로 내 서재로 이동
