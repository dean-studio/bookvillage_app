## 1. 공통 설계 기준

### 1.1 반응형 브레이크포인트

| 구분 | 너비 | 대상 기기 | 비고 |
|------|------|-----------|------|
| Mobile | < 768px | 스마트폰 | 주민 주 사용 환경 |
| Tablet | 768px ~ 1023px | 태블릿 | 관리자 주 사용 환경 |
| PC | >= 1024px | 데스크톱 | 관리자 대시보드 최적화 |

### 1.2 공통 레이아웃 구조

```
+--------------------------------------------------+
| Header (AppBar)                                  |
|  [로고: 책빌리지]          [사용자명] [로그아웃]    |
+--------------------------------------------------+
| Navigation (주민: BottomNav / 관리자: Sidebar)    |
+--------------------------------------------------+
| Main Content Area                                |
|                                                  |
+--------------------------------------------------+
```

**주민 영역 (Mobile 우선)**
- 하단 탭 네비게이션: 도서검색 | 내 서재
- Header: 로고 + 사용자명 + 로그아웃

**관리자 영역 (Tablet/PC 우선)**
- 좌측 사이드바 네비게이션 (Tablet에서는 접이식)
- 사이드바 메뉴: 대시보드 / 도서등록 / 대출 / 반납 / 관리

### 1.3 공통 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| `AppHeader` | 상단 바 (로고, 사용자 정보, 로그아웃) |
| `BottomNav` | 주민용 하단 탭 네비게이션 |
| `AdminSidebar` | 관리자용 좌측 사이드바 |
| `BookCard` | 도서 카드 (표지, 제목, 저자, 대출가능 뱃지) |
| `BarcodeScanner` | 카메라 바코드 스캔 모달 |
| `Modal` | 범용 모달 (확인/취소) |
| `Toast` | 알림 토스트 메시지 |
| `LoadingSpinner` | 로딩 인디케이터 |
| `EmptyState` | 데이터 없음 안내 |
| `Badge` | 상태 뱃지 (대출가능/대출중/연체) |

### 1.4 색상 체계

| 용도 | 색상 | 설명 |
|------|------|------|
| Primary | #2563EB (Blue 600) | 주요 액션 버튼, 링크 |
| Success | #16A34A (Green 600) | 대출가능, 반납완료 |
| Warning | #F59E0B (Amber 500) | 연체 경고 |
| Danger | #DC2626 (Red 600) | 에러, 삭제 |
| Neutral | #6B7280 (Gray 500) | 비활성, 보조 텍스트 |

## 2. /login - 로그인 페이지

### 2.1 사용자 플로우

```
[앱 접속] --> [로그인 폼 표시]
  ├─ 기존 회원 --> [전화번호 + PIN 입력] --> [로그인 성공] --> [/books 이동]
  │                                      └─ [실패] --> [에러 메시지]
  └─ 신규 회원 --> [회원가입 탭 전환]
       --> [전화번호 + 이름 + 동/호수 + PIN 설정]
       --> [가입 완료] --> [/books 이동]
```

### 2.2 컴포넌트 구조

```
LoginPage
├── LoginForm
│   ├── PhoneInput (전화번호 입력, 하이픈 자동 포맷)
│   ├── PinInput (4자리 숫자 입력, 마스킹)
│   └── SubmitButton
├── RegisterForm (탭 전환 시)
│   ├── PhoneInput
│   ├── NameInput
│   ├── AddressInput (동 선택 + 호수 입력)
│   ├── PinInput (4자리 설정)
│   ├── PinConfirmInput (4자리 확인)
│   └── SubmitButton
└── TabSwitcher ("로그인" | "회원가입")
```

### 2.3 와이어프레임

**Mobile (< 768px)**
```
+--------------------------------+
|         책빌리지 로고            |
|      아파트 스마트 도서관         |
+--------------------------------+
|                                |
|  [로그인]  [회원가입]  <-- 탭    |
|  ──────────────────            |
|                                |
|  전화번호                       |
|  +----------------------------+|
|  | 010-1234-5678              ||
|  +----------------------------+|
|                                |
|  비밀번호 (4자리)               |
|  +----------------------------+|
|  | ● ● ● ●                   ||
|  +----------------------------+|
|                                |
|  +----------------------------+|
|  |        로그인하기            ||
|  +----------------------------+|
|                                |
+--------------------------------+
```

**Tablet/PC (>= 768px)**
```
+----------------------------------------------------------+
|                                                          |
|   +-------------------+   +---------------------------+  |
|   |                   |   |                           |  |
|   |    책빌리지 로고    |   |   [로그인]  [회원가입]     |  |
|   |    일러스트        |   |   ────────────────        |  |
|   |                   |   |                           |  |
|   |                   |   |   전화번호                 |  |
|   |                   |   |   [010-1234-5678      ]   |  |
|   |                   |   |                           |  |
|   |                   |   |   비밀번호 (4자리)         |  |
|   |                   |   |   [● ● ● ●            ]   |  |
|   |                   |   |                           |  |
|   |                   |   |   [    로그인하기      ]   |  |
|   |                   |   |                           |  |
|   +-------------------+   +---------------------------+  |
|                                                          |
+----------------------------------------------------------+
```

### 2.4 주요 인터랙션

- 전화번호: 숫자만 입력 가능, 010-XXXX-XXXX 자동 포맷
- PIN: 4자리 숫자, 입력 시 마스킹(●) 처리
- 로그인 실패: 폼 상단에 인라인 에러 메시지 ("전화번호 또는 비밀번호가 올바르지 않습니다")
- 로그인 성공: role에 따라 주민은 `/books`, 관리자는 `/admin`으로 리다이렉트

## 3. /books - 도서 검색 페이지

### 3.1 사용자 플로우

```
[페이지 진입] --> [전체 도서 목록 표시]
  ├─ [검색어 입력] --> [실시간 필터링]
  ├─ [대출가능 필터 토글] --> [목록 필터링]
  └─ [도서 카드 탭] --> [/books/[id] 상세 이동]
```

### 3.2 컴포넌트 구조

```
BooksPage
├── SearchBar
│   ├── SearchInput (제목/저자 검색)
│   └── FilterToggle ("대출 가능만 보기" 체크박스)
├── BookGrid
│   └── BookCard (반복)
│       ├── CoverImage (표지 썸네일)
│       ├── BookTitle
│       ├── BookAuthor
│       └── AvailabilityBadge ("대출가능" | "대출중")
├── EmptyState (검색 결과 없음)
└── LoadMoreButton (페이지네이션 또는 무한스크롤)
```

```
[마이페이지 진입] --> [현재 대출 현황 표시]

  ├─ [반납 완료 도서]
  │    ├─ [독서 퀴즈 풀기] --> [퀴즈 모달]
  │    └─ [독서록 쓰기] --> [독서록 작성 모달]
```

```
│       ├── BookTitle
│       ├── ReturnedDate

│   ├── RatingInput (별 1~5개)
```

- 독서록 버튼: 해당 도서에 독서록을 아직 작성하지 않은 경우에만 활성
- 퀴즈 결과: 정답/오답 즉시 피드백 (정답 시 축하 메시지, 오답 시 정답 표시)
- 독서록: 별점 필수, 감상평 최소 10자 이상

## 6. /admin - 관리자 대시보드

### 6.1 사용자 플로우


## 11. /admin/deletions - 도서 삭제 내역

### 11.1 기능

- 삭제된 도서의 감사 로그를 조회하는 전용 페이지
- 사이드바 메뉴에서 "삭제 내역"으로 접근

### 11.2 표시 정보

| 컬럼 | 설명 |
|------|------|
| 도서명 | 삭제된 도서 제목 |
| 저자 | 삭제된 도서 저자 |
| 바코드 | 삭제된 도서 바코드 |
| 삭제자 | 삭제를 수행한 관리자 이름 |
| 삭제일시 | 삭제 시각 |

### 11.3 UI

- 데스크탑: 테이블 형태
- 모바일: 카드 리스트
- 페이지네이션 (20건 단위)

## 12. /admin/books/shelf/[shelfName] - 서재별 도서 관리

### 12.1 기능

- 특정 서재에 속한 도서 CRUD
- 바코드 스캔으로 도서 등록 (카카오 API 자동완성)
- 하이브리드 바코드: ISBN 중복 시 자체 바코드 등록 안내
- 도서 편집/삭제 (삭제 시 대출 중 차단 + 대여 이력 경고)

### 12.2 UI

- 데스크탑: 테이블 + 인라인 등록 폼
- 모바일: 카메라 바코드 스캔 + 카드 리스트
- 페이지네이션 (15건 단위)
