# 랭킹

기간별 다양한 랭킹을 조회합니다.

## 날짜 필터

![img_42.png](img_42.png)

시작일~종료일을 직접 설정하거나 빠른 선택 버튼을 사용합니다.

| 버튼 | 기간 |
|------|------|
| 최근 1주 | 7일 |
| 최근 1개월 | 30일 |
| 최근 3개월 | 90일 |
| 최근 6개월 | 180일 |
| 올해 | 1월 1일부터 |

## 랭킹 탭

<!-- ![랭킹 탭](../images/admin-rankings-tabs.png) -->

4개 탭이 하나의 카드 안에 통합되어 있습니다.

### 다독왕

가장 많이 대출한 주민 순위입니다. 클릭 시 주민 상세로 이동합니다.

### 인기 도서

가장 많이 대출된 도서 순위입니다. 클릭 시 도서 상세로 이동합니다.

![img_43.png](img_43.png)
### 독서록

독서록을 많이 작성한 주민 순위입니다.

### 젤리

해당 기간 동안 젤리를 가장 많이 획득한 주민 순위입니다.

::: info 순위 배지
1위는 금색, 2위는 은색, 3위는 동색 배지로 표시됩니다.
:::
