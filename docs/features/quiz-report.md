# 퀴즈 & 독서록

반납 완료된 도서에 대해 퀴즈 풀기와 독서록 작성이 가능합니다.

## 퀴즈 풀기

관리자가 등록한 퀴즈가 있는 도서에서 활성화됩니다.

<!-- ![퀴즈 풀기](../images/quiz-solve.png) -->

- 객관식 문제 1문항
- 정답/오답 결과 즉시 확인
- 1인 1회 제한 (이미 푼 퀴즈는 비활성화)

::: tip 젤리 획득
퀴즈 정답 시 **+3 젤리**가 지급됩니다.
:::

## 독서록 쓰기

반납 완료된 도서에서 독서록을 작성할 수 있습니다.

<!-- ![독서록 작성](../images/report-write.png) -->

### 작성 항목

- **별점**: 1~5점 선택
- **감상문**: 10자 이상 작성

### 작성 완료

독서록이 등록되면 토스트 메시지로 확인합니다.
이미 작성한 도서는 "작성 완료"로 표시됩니다.

<!-- ![독서록 완료](../images/report-done.png) -->

::: tip 젤리 획득
독서록 작성 시 **+10 젤리**가 지급됩니다.
:::

## 미작성 독서록 권유

알림 화면에서 미작성 도서에 대한 권유 카드가 표시됩니다.
터치하면 반납완료 페이지로 이동합니다.
