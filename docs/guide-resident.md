# 책빌리지 주민 사용 가이드

## 1. 회원가입 / 로그인

### 첫 이용 (회원가입)
1. 앱에 접속하면 로그인 화면이 나타납니다
2. **이름** 입력 → **휴대폰 번호** 입력 → **동** 입력 → **호** 입력 → **PIN 4자리** 설정
3. 모든 숫자는 화면의 큰 숫자패드로 입력합니다
4. 각 단계에서 "이전" 버튼으로 돌아갈 수 있습니다

### 로그인
1. **휴대폰 번호** 입력
2. **PIN 4자리** 입력
3. 자동으로 내 서재로 이동합니다

---

## 2. 대여하기 (`대여` 탭)

대여 화면 상단에는 **도서관 로고**와 **아파트 이름**이 표시됩니다.

### 바코드 스캔으로 대여
1. 하단 네비게이션에서 **대여** 탭을 선택합니다
2. 화면 중앙의 **바코드 스캔** 버튼을 누릅니다
3. 카메라가 열리면 책 뒷면의 **바코드**를 비춥니다
4. 도서 정보가 화면에 표시됩니다
5. **대여하기** 버튼을 눌러 대출을 완료합니다
6. 대출 완료 시 **+5 젤리**가 자동 지급됩니다

### 바코드 수동 입력
- 카메라를 사용할 수 없는 경우, 화면 하단의 입력칸에 바코드 번호를 직접 입력할 수 있습니다

### 인기 검색어
- 대여 화면에 다른 주민들이 많이 검색한 도서가 롤링 형태로 표시됩니다

---

## 3. 도서 검색 (`도서` 탭)

1. 하단 네비게이션에서 **도서** 탭을 선택합니다
2. 상단 검색창에 **도서명** 또는 **저자**를 입력합니다
3. **전체** / **대출 가능** 필터로 대출 가능한 도서만 볼 수 있습니다
4. 도서를 터치하면 **도서 상세** 화면으로 이동합니다

### 도서 상세
- 표지, 제목, 저자, 출판사, 소개글 등 상세 정보를 볼 수 있습니다
- **대출 가능** 상태일 때만 **서가 위치 맵**이 표시됩니다
- "이 책을 본 사람이 본 도서" **추천 도서**가 하단에 표시됩니다

---

## 4. 내 서재 (`내서재` 탭)

내 서재 메인 화면에는 다음 정보가 표시됩니다:
- **이름**, **동호수**
- **알림** 아이콘 (미읽은 알림 수 배지)
- **보유 젤리** 잔액
- 연체 도서가 있으면 **경고 배너** 표시

### 4-1. 대여중 도서
- 현재 대출 중인 도서 목록
- 각 도서에 반납 기한까지 남은 일수가 **D-숫자** 배지로 표시
- 연체 중이면 빨간색 **연체** 배지 표시
- 도서를 터치하면 도서 상세로 이동

### 4-2. 반납완료 도서
반납된 도서 목록이 표시됩니다. 각 도서에서:

#### 퀴즈 풀기
- 관리자가 등록한 퀴즈가 있으면 **퀴즈 풀기** 버튼이 활성화됩니다
- 객관식 문제를 풀고 정답/오답 결과를 확인합니다
- 정답 시 **+3 젤리** 지급

#### 독서록 쓰기
- **독서록 쓰기** 버튼을 누르면 인라인 폼이 펼쳐집니다
- **별점** (1~5점) 선택 + **감상문** (10자 이상) 작성
- 등록 완료 시 **+10 젤리** 지급
- 이미 작성한 도서는 "작성 완료"로 비활성화

### 4-3. 알림
- **반납 예정 알림**: 반납일 3일 전부터 알림
- **당일 반납 알림**: 반납일 당일 알림
- **연체 알림**: 연체 시 빨간색 경고 카드
- **독서록 권유**: 미작성 도서에 대해 "독서록을 써보세요!" 카드 표시 (터치 시 반납완료 페이지로 이동)

### 4-4. 내 젤리
- **보유 젤리** 잔액과 **총 획득량** 표시
- 젤리 획득 방법 안내: 대출 +5 / 반납 +5 / 독서록 +10 / 퀴즈 정답 +3
- **이력**: 날짜별 젤리 획득/차감 내역 (사유, 도서명 포함)

### 4-5. 로그아웃
- 화면 하단의 **로그아웃** 버튼으로 로그아웃
