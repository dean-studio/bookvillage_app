# 🏡 책빌리지 (Book Village)

**아파트 스마트 작은도서관 관리 및 독서 커뮤니티 시스템**

수기나 엑셀로 관리되던 아파트 도서관 업무를 전산화하여 사서(자원봉사자)의 업무 부담을 줄이고, 주민들에게는 스마트폰으로 편리한 도서 검색 및 대여 현황 조회, 그리고 독서 후 활동(퀴즈/독서록) 서비스를 제공합니다.

## ✨ 핵심 기능 및 특징

- **📱 초간편 로그인:** 이메일 가입의 번거로움 없이 **'휴대폰 번호 + 4자리 비밀번호'**로 가입 및 로그인.
- **🏷️ 하이브리드 바코드 시스템:** 출판사 기본 ISBN 바코드를 활용하되, 복본(같은 책 2권 이상)일 경우에만 아파트 자체 바코드 스티커를 부착.
- **⚡ 스마트 도서 등록:** 바코드 스캔 시 **카카오 도서 검색 API**로 도서 정보가 자동 완성되며, 체계적인 서가 그룹(예: 어린이 코너 > A-3) 지정 가능.
- **📚 스마트 반납 및 위치 안내:** 반납 스캔 시 즉시 반납 처리되며, 화면에 해당 책을 꽂아야 할 **정확한 서가 위치**가 팝업되어 사서의 정리 동선 최소화.
- **⏰ 연체 관리 자동화:** 연체 7일 차, 30일 차에 시스템이 자동으로 주민 휴대폰에 **카카오 알림톡** 발송.
- **🏆 독서 활동 보상 체계:** 책을 정상 반납 완료한 주민에게만 **'독서 퀴즈 풀기'** 및 **'독서록 쓰기'** 메뉴 활성화.
- **📊 스마트 통계 대시보드 🌟(신규):** 관리자가 특정 기간(예: 이번 달)을 설정하여 '다독왕(대출 1위 주민)', '인기 도서 랭킹' 등을 원클릭으로 추출 가능.

## 🛠️ 기술 스택 (Tech Stack)

- **Frontend / Backend:** Next.js (App Router)
- **Database / Auth:** Supabase (PostgreSQL)
- **Deployment:** Vercel
- **External API:** 카카오 도서 검색 API, 카카오 알림톡 API (솔라피/알리고)

## 🗄️ 데이터베이스 구조 (Supabase)

1. **`profiles` (주민):** `id`, `phone_number`(ID), `name`, `dong_ho`, `role`
2. **`books` (도서):** `id`, `barcode`, `title`, `author`, `cover_image`, `location_group`, `location_detail`, `is_available`
3. **`rentals` (대여 기록):** `id`, `book_id`, `user_id`, `rented_at`, `due_date`, `returned_at`
4. **`quizzes` (독서 퀴즈):** `id`, `book_id`, `question`, `options`, `answer`
5. **`book_reports` (독서록):** `id`, `book_id`, `user_id`, `rating`, `review`

## 🗺️ 사이트맵 (Sitemap)

### 👤 일반 주민 영역 (Mobile/PC)

- `/login`: 휴대폰 번호 및 PIN 번호 로그인
- `/books`: 도서 검색, 필터링(대출 가능), 도서 상세조회(서가 위치 직관적 안내)
- `/mypage`: 내 서재 (현재 대출 현황, 독서록 작성, 퀴즈 풀기)

### 👩‍💻 관리자 영역 (Tablet/PC 최적화)

- `/admin` (대시보드): 📊 **기간별 통계 추출 (이달의 다독왕, 베스트 대출 도서, 우수 독서록 작성자 등)**
- `/admin/books/new`: 스마트 도서 등록 (ISBN 스캔 ➡️ 서가 지정)
- `/admin/checkout`: 대출 처리 (주민 선택 ➡️ 도서 스캔)
- `/admin/return`: 반납 처리 (도서 스캔 ➡️ 즉각적인 위치 확인)
- `/admin/manage`: 도서별 퀴즈 등록 및 연체 알림 관리
